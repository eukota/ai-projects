# AI Principles

## Managed Context

Project context should be durable, addressable, and portable. It should not depend on one chat product's private memory or UI folder model.

## Source of Truth

- Design source of truth: this repository.
- Implementation source of truth: separate implementation repositories.
- Cross-linking source of truth: project docs in this repository.

## Agent Collaboration

Agents should be able to enter a project, read the current context, understand prior decisions, and continue work without reconstructing intent from scattered conversations.

## Explicit Boundaries

Architecture notes, planning, decision records, experiments, and references belong here.

Running applications, libraries, CLIs, services, generated builds, and dependency trees belong elsewhere.
