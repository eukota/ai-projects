# Permissions and Security

AIOS needs real access control because agents can read memory, use tools, execute workflows, and affect external systems.

## Subjects

- Users
- Agents
- Teams
- Services

## Resources

- Memory scopes
- Tools
- Files
- Repositories
- Workflows
- APIs
- Devices

## Actions

- Read
- Write
- Execute
- Delegate
- Approve
- Delete
- Share

## Principles

- Least privilege
- Revocable access
- Scoped credentials
- Full audit trail
- Human approval for high-risk actions

## Human Controls

Humans remain in charge.

Roles:

- Operator
- User
- Reviewer
- Admin
- Teacher
- Collaborator

Controls:

- Approve actions
- Inspect memory
- Terminate workflows
- Edit goals
- Change policies
- Override outputs
- Tune routing preferences
