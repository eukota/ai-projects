# Code Proficiency

Traditional coding skills. Same as non-AI interviews—but evaluated in an AI-assisted context.

---

## Non-Passing

**Demonstrated knowledge:**
- Generates code with AI but can't explain it
- Can't navigate their own generated code
- Asked "Where are the defaults set?", they can't find them
- No coding fundamentals or forgot them

**Interviewer observations:**
- Candidate doesn't understand what the code does
- Can't walk through the logic
- Didn't write it and can't own it
- Fundamental gaps in coding ability

**Do they need training?** Yes

**Where to start:** Phase B (fundamentals), then Phase C

---

## Entry

**Demonstrated knowledge:**
- Can code; has the fundamentals
- Code might be messy
- Can navigate and understand their code
- Can explain what the code does

**Interviewer observations:**
- Candidate knows how to code
- Can read and walk through code
- Understands basic structure
- Can ask for help/clarification

**Do they need training?** Depends on other rubric scores

**Where to start:** Phase C, Levels 4-6

---

## Mid

**Demonstrated knowledge:**
- Knows how to code well
- Easily browses and understands code
- Can walk through code and explain it
- Can see what's missing or could be better
- Has insights about the code

**Interviewer observations:**
- Candidate demonstrates strong coding fundamentals
- Can explain design choices
- Sees potential issues or improvements
- Code quality is solid

**Do they need training?** No

**Where they are:** Phase C, Levels 6-8

---

## Senior

**Demonstrated knowledge:**
- Deep understanding of code
- Talks about code with nuance and depth
- Explains trade-offs of different functions/approaches
- Understands *why* AI chose certain approaches
- Can discuss performance, maintainability, extensibility

**Interviewer observations:**
- Candidate demonstrates expert-level coding understanding
- Explains design decisions clearly
- Asks good follow-up questions
- Shows mastery of their language/domain

**Do they need training?** No

**Where they are:** Phase C, Levels 8-11; Phase D

---

## Key Point

**Code Proficiency is about traditional engineering skills.** It's not different from non-AI interviews. The difference is everything else (how they work WITH AI).
