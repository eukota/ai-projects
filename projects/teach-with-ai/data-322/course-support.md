# DATA 322 Course Support Ideas

## Before Class

- Generate lesson outlines from instructor notes.
- Prepare examples at multiple difficulty levels.
- Identify likely misconceptions.
- Build quick checks for understanding.

## During Class

- Let students ask individualized questions.
- Provide alternate explanations.
- Surface confusion clusters to the instructor.
- Support live coding, data analysis, or concept practice.

## After Class

- Help students review concepts.
- Generate practice questions.
- Summarize common issues for instructor follow-up.
- Support project planning and reflection.

## Assignment Support

The agent should help students understand requirements, plan work, debug process, and reflect on results without bypassing learning objectives.
