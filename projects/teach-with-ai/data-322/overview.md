# DATA 322 AI Course Pilot

DATA 322 at Cal Poly Humboldt is the first concrete pilot for Teach With AI.

This track should translate the broader Teach With AI idea into course-specific support: tutoring, assignment scaffolding, instructor analytics, and safe workflows for students.

## Current Status

Seed concept.

## Course Context Needed

- Official course title
- Catalog description
- Syllabus
- Weekly topic outline
- Assignment types
- Tooling expectations
- Grading policies
- Academic integrity policy
- Student privacy constraints

## Relationship to Teaching AI

Teaching AI is the broader mission.

DATA 322 is a concrete implementation path and proving ground.
