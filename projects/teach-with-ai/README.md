# Teach With AI

## Summary

Teach With AI is the project for using AI as part of the instructional operating model for existing subjects.

This is different from teaching people about AI. The goal is to use AI to help teach a course or subject better.

## First Pilot

[DATA 322 at Cal Poly Humboldt](data-322/overview.md)

DATA 322 is the first concrete example because it is real, bounded, and credible. A successful pilot can demonstrate how AI-native course design works in practice.

## Focus

- AI tutors
- Course-aware assistants
- Student support
- Instructor dashboards
- Adaptive explanations
- Assignment scaffolding
- Misconception tracking
- Safe classroom workflows
