# Interviewer Rubric

Evaluation framework for assessing candidates in AI-assisted technical interviews.

## Status

*Legacy materials exist in `teaching-ai/phases/d/ai-solution-strategy/` — to be integrated and updated*

## Purpose

This section contains:
- Rubric for evaluating AI awareness, code proficiency, AI proficiency
- Guidance on assessing how candidates approach problem-solving with AI
- Evaluation criteria for Requirements gathering, Implementation, and Testing phases
- How to spot candidates who understand AI fundamentals vs. those still learning

## For Interviewers

Use this rubric when:
- Evaluating candidate performance during AI-assisted interviews
- Assessing how well candidates direct and leverage AI
- Determining if candidates blindly accept AI output or think critically
- Understanding the different levels of AI competency (Entry → Mid → Senior)

## Integration with Mock Interview Agent

When the mock interview agent is built, it will use this rubric to automatically evaluate candidate performance and provide feedback.
