# Mock Interview Agent

Interactive AI agent for practicing AI-assisted technical interviews.

## Status

*Planned — Not yet implemented*

## Vision

An AI agent that:
- Simulates an interviewer asking coding/design problems
- Responds to candidate questions in real-time
- Evaluates performance using the interviewer rubric
- Provides feedback on how well you demonstrated AI competency
- Can be run locally or as part of the candidate preparation workflow
