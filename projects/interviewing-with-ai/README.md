# Interviewing with AI

Framework, tools, and guidance for technical interviews in an AI-assisted context.

This project contains everything needed to prepare for, conduct, and evaluate AI-assisted technical interviews.

## Structure

### Candidate Guide

How to perform well in AI-assisted technical interviews. Learn the strategy for solving problems with AI, demonstrating competency, and managing interview constraints.

- **AI Solution Strategy** — Practical techniques for working with AI during problem-solving
- **Interview Preparation** — How to communicate your AI skills and perform well under interview conditions
- **Examples** — Worked examples at Entry and Senior complexity levels

**Location:** `candidate-guide/`

### Mock Interview Agent

Interactive AI agent for practicing interviews. The agent simulates an interviewer, responds to your questions, and evaluates your performance using the interviewer rubric.

*Status: Planned*

**Location:** `mock-interview-agent/`

### Interviewer Rubric

Evaluation framework for assessing candidates in AI-assisted technical interviews. What interviewers look for and how to evaluate performance across different dimensions.

*Status: Existing (legacy materials in teaching-ai/phases/d/ai-solution-strategy/)*

**Location:** `interviewer-rubric/`

---

## How to Use

**For candidates:** Start with `candidate-guide/` to learn the strategy and examples.

**For interviewers:** Refer to `interviewer-rubric/` for evaluation guidance.

**For practice:** Use `mock-interview-agent/` to run mock interviews and get feedback (when available).
